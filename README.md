# Scientific Calculator

A modern and responsive Scientific Calculator built with HTML, CSS, and JavaScript. This application supports both basic arithmetic operations and scientific functions while providing a clean user interface, calculation history, keyboard support, and light/dark mode switching.

## 🚀 Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal calculations

### Scientific Functions
- Sine (`sin`)
- Cosine (`cos`)
- Tangent (`tan`)
- Square Root (`√`)

### Additional Features
- Calculation History
- Keyboard Support
- Dark Mode
- Light Mode
- Delete Last Character
- Clear Display (AC)
- Error Handling
- Responsive Design

##  Usage

### Mouse Input

1. Click number buttons to enter values.
2. Select an operator or scientific function.
3. Click `=` to calculate.
4. Use:
   - `AC` to clear the display.
   - `DEL` to remove the last character.

### Keyboard Shortcuts

| Key | Action |
|------|---------|
| 0-9 | Enter numbers |
| + - * / | Arithmetic operations |
| Enter | Calculate result |
| Backspace | Delete last character |
| Escape | Clear display |
